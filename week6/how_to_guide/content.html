Purpose:

This API is the replacement for the official netflix search API. In late 2014 the Official Netflix public API was shutdown. Despite the name This Netflix API is NOT a random movie generator but similar to a movie search engine that works via get requests.

How to make an API request?

Requests can be made via Get requests to the following url:

http://netflixroulette.net/api/api.php?

Parameters for GET requests:

  • Genre (string)
  • Movies (boolean)
  • TV (boolean)
  • LowRating (float 1-5)
  • HighRating (float 1-5)
  • Director (string)
  • Keyword (string)
  • Year (int)
  • type (string)

Sample code: (JavaScript)

var API_URL = http://netflixroulette.net/api/api.php?; //note the question mark at the end

var title_value = "ip man";
var predicate = "title=" + URLEncodeComponent(title_value);//making string readable for get requests

var searchURL = API_URL + predicate;

console.log(searchURL);
// output should be: http://netflixroulette.net/api/api.php?title=ip%20man

description: display movie data from title alone.
Disred output: http://netflixroulette.net/api/api.php?title=ip%20man
